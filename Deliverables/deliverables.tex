Il presente documento intende fornire i puntatori ai documenti richiesti, in quanto prodotti attesi di progetto,
dall'azienda proponente Vimar S.p.A. per il collaudo e consegna finale di quanto realizzato nel progetto View4Life.
Nello specifico sono riportati:
\begin{itemize}
    \begin{frame}
    \item il puntatore al documento sull'architettura realizzata tramite modello C4 (diagramma di contesto, digramma di container);
    \item il puntatore ai bozzetti grafici relativi alla progettazione di design dell'applicativo web responsive;
    \item il puntatore al documento di preventivo relativo alle spese stimate dell'applicativo web in Cloud AWS;
    \item il puntatore al documento contenente i casi d'uso per l'utilizzo dell'applicativo web;
    \item il puntatore alla documentazione web delle API backend realizzate;
    \item il puntatore alla documentazione web di tutti i test realizzati e report, con relativo coverage e risultato;
    \item il puntatore al video dimostrativo dell'applicativo web in funzione;
    \item il puntatore al manuale di utilizzo dell'applicativo web per gli utenti.
\end{itemize}
